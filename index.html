ANOVA + Tukey (por grupos)

Pegá tu tabla, definí cómo se segmenta el experimento y descargá un Excel con tu misma data + resultados.

Esto está pensado para análisis rápidos de ensayos: vos definís el valor, el tratamiento y las columnas que forman cada grupo. El backend corre el análisis y te devuelve el archivo listo.

1) Pegar datos

Acepta TSV (copiado desde Excel) o CSV. La primera fila debe ser encabezados.

2) Configurar el análisis

Ej: assessment_value

Ej: treatment

Por defecto: 0.05

Elegí todas las columnas que “parten” el experimento (momento, trial, parte evaluada, etc). Cada combinación define un grupo con su propio ANOVA.
